How to work as a group:
*cmd: | git branch |                     ---> list the branch name created with that repo.
*cmd: | git branch -c <YourBranchName> | ---> Create branch in your name.
*cmd: | git status |                     ---> show changes done in the file.

Edit the file you want to edit.

*cmd: | git add <fileName> |             ---> add the edited file to the repo
*cmd: | git add <.> |                    ---> add the all files to the repo
*cmd: | git commit -m "<Message>" |      ---> commit files to the repo
*cmd: | git push origin head |           ---> Push the files to git

This can be done manually by loggin in git hub
*cmd: | git checkout main |              ---> to goto the main file of the repo
*cmd: | git merge <brachname> |          ---> merge the branch to the main file
